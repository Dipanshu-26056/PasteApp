# CodePaste

A clean code/notes paste app built with React, Redux Toolkit, React Router, and Tailwind CSS. All data is saved in `localStorage`.

## Features
- Create and edit pastes with a code editor (line numbers, tab support)
- View all pastes with search
- Copy, Download, Share, Edit, Delete each paste
- View individual pastes in read-only mode
- Persistent storage via localStorage
- Toast notifications

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Stack
- React 18
- Redux Toolkit (state management)
- React Router v6 (routing)
- React Hot Toast (notifications)
- Tailwind CSS (styling)
- Vite (build tool)
